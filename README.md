# 🎮 Last Light (Working Title)

A small, polished, systems-driven survival game built in Unity.

------------------------------------------------------------------------

## 🧭 Overview

**Last Light** is a top-down survival game where the player must gather
resources during the day and survive dangerous nights using light as
their only protection.

> **Core Idea:** Light is safety. Darkness is danger.

------------------------------------------------------------------------

## 🔁 Core Gameplay Loop

-   Explore during the day\
-   Gather resources\
-   Craft tools and light sources\
-   Survive the night\
-   Repeat and improve

------------------------------------------------------------------------

## 🔥 Key Features

-   🌗 Day/Night cycle with tension-based gameplay\
-   🔥 Light system (core mechanic)\
-   🍗 Hunger & stamina survival systems\
-   🎒 Inventory & crafting\
-   👾 Enemy AI that reacts to light\
-   ✨ Juice & polish (VFX, SFX, feedback)

------------------------------------------------------------------------

## 🛠️ Built With

-   Unity (C#)
-   ScriptableObjects for data-driven design
-   Modular system architecture

------------------------------------------------------------------------
